3 Decisiones del día

  • Toda automatización ejecuta comandos estáticos; lo dinámico (fechas, slugs, tokens) se resuelve dentro de un script — así no se bloquea el "Siempre permitir".
  • Nada corre con el PC apagado: las corridas perdidas no se recuperan solas; se necesita una Tarea Programada de Windows para cobertura real.
  • Idempotencia con marcador por día / timestamp en las tareas de un solo disparo, para no enviar dos veces aunque haya 3 ventanas de seguridad.
  • La plataforma de videos médicos sale a producción detrás de un gate de autenticación de dueño, no público.
